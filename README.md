# ArisuAutoScript

**蔚蓝档案自动化脚本**

**| [English](README.en.md) | 简体中文 |**

![gui_cn.png](docs/resources/gui_cn.png)

## 功能

当前脚本还在活跃开发中，已经实现的功能有：

- [x] **咖啡厅** 领取奖励 / 互动 / 第二咖啡厅
- [x] **公会** 领取体力
- [x] **邮箱** 领取奖励
- [x] **战术对抗赛** 领取奖励 / 自动战斗

_目前仅支持日服。_
